Genshin Ascension Cards

Proyecto final del módulo de “Fullstack”. La idea era crear una aplicación web capaz de generar unas tarjetas según el personaje elegido. Estas tarjetas nos brindarían información sobre los materiales necesarios para subir de nivel a dichos personajes. Los personajes son parte del videojuego llamado “Genshin Impact”. Decidí implementar también un botón de descarga para poder guardar la tarjeta generada en el dispositivo.

¿Qué herramientas utilicé?

La página web se diseñó utilizando HTML5, CSS3, Javascript y jQuery. La información se subió a una base de datos facilitada por Heroku y se conectó a la web con php. El código está subido a un repositorio de Github y sincronizado para que cualquier cambio realizado sea efectivo casi al instante.
