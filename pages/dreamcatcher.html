Dreamcatcher

Final project of the "Wordpress Course". Our goal was to create a webpage dedicated to the Kpop girl group "Dreamcatcher". This website has information about the group members, their discography, their history and you can also suscribe to a "Newsletter" to stay tuned for new updates.

What tools did I use?

This website was made by using "Elementor Pro" as a theme builder. This project was designed from zero and it had a phase of creative design (moodboard, prototyping, etc.) and a phase of building using Wordpress. We used plugins like "Loginpress (used to customize the access to the administration panel of the website),"WPS Child Theme Generator" (use to create child themes), "WP-Optimize" (for web and image optimization), etc.

More about the project

The project used a lot of plugins to improve its functionality. "WPS Hide Login" was used to hide the administration panel of the website (getting rid of the famous "wp-admin" and hiding it from the visitors). "Wordfence" was used as an antivirus and antimalware plugin. "All-in One WP-Migration" was also used to create backups in case we need them (or even to migrate the website).
